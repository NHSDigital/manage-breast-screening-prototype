Details

An extended version of the NHS <details> component. Adds support for a subtitle beneath the summary, a contents summary line, and an optional status tag overlay.

Macro: appDetails(params)

Basic

Behaves identically to the standard NHS details component.

{{ appDetails({ summaryText: "Help with this question", text: "This is the body text of the details element." }) }} 
{% raw %}{{ appDetails({
  summaryText: "Help with this question",
  text: "This is the body text of the details element."
}) }}{% endraw %}

With subtitle

Use subtitle to show secondary text beneath the summary label — useful for showing a date range or count.

{{ appDetails({ summaryText: "Previous mammogram", subtitle: "St Thomas' Hospital, 2022", text: "Full details about the previous mammogram would appear here." }) }} 
{% raw %}{{ appDetails({
  summaryText: "Previous mammogram",
  subtitle: "St Thomas' Hospital, 2022",
  text: "..."
}) }}{% endraw %}

With contents summary

Use contentsSummary (text) or contentsSummaryHtml to show a summary of what's inside. This appears on the same line as the subtitle.

{{ appDetails({ summaryText: "Medical history", contentsSummary: "3 items", html: "

Three items of medical history would appear here.

" }) }} 
{% raw %}{{ appDetails({
  summaryText: "Medical history",
  contentsSummary: "3 items",
  html: "
...
"
}) }}{% endraw %}

Open by default

Use open: true to render the details element expanded.

{{ appDetails({ summaryText: "Expanded by default", open: true, text: "This content is visible without interaction." }) }} 
{% raw %}{{ appDetails({
  summaryText: "Expanded by default",
  open: true,
  text: "..."
}) }}{% endraw %}

With status tag

Use status (a status string passed through the toTag filter) to overlay a status tag on the summary. Used to show the completion state of a section.

{{ appDetails({ summaryText: "Consent", status: "complete", text: "Consent has been recorded." }) }} 
{% raw %}{{ appDetails({
  summaryText: "Consent",
  status: "complete",
  text: "..."
}) }}{% endraw %}
