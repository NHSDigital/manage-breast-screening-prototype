Forward link

A directional link pointing forward — the visual counterpart to the NHS back link. Renders as an anchor by default.

Macro: appForwardLink(params)

As a link (default)

{{ appForwardLink({ text: "Continue to next step", href: "#" }) }} 
{% raw %}{{ appForwardLink({
  text: "Continue to next step",
  href: "#"
}) }}{% endraw %}

As a button

Use element: "button" to render as a <button> for use inside forms where no href is available.

{{ appForwardLink({ text: "Continue", element: "button" }) }} 
{% raw %}{{ appForwardLink({
  text: "Continue",
  element: "button"
}) }}{% endraw %}

With custom text

{{ appForwardLink({ text: "Go to participant record", href: "#" }) }} 
{% raw %}{{ appForwardLink({
  text: "Go to participant record",
  href: "#"
}) }}{% endraw %}
