Summary card

A summary card with a title bar and freeform content area. Uses the Nunjucks caller() pattern — wrap the content in a {% raw %}{% call appSummaryCard(...) %}...{% endcall %}{% endraw %} block.

Macro: appSummaryCard(params)

Basic

{% call appSummaryCard({ title: { text: "Previous mammogram" } }) %} {{ appSummaryList({ rows: [ { key: { text: "Date" }, value: { text: "June 2022" } }, { key: { text: "Location" }, value: { text: "St Thomas' Hospital" } } ] }) }} {% endcall %} 
{% raw %}{% call appSummaryCard({
  title: { text: "Previous mammogram" }
}) %}
  {{ appSummaryList({ rows: [...] }) }}
{% endcall %}{% endraw %}

With single action

{% call appSummaryCard({ title: { text: "Implant — left breast" }, actions: { items: [ { text: "Edit", href: "#", visuallyHiddenText: "implant details" } ] } }) %} {{ appSummaryList({ rows: [ { key: { text: "Type" }, value: { text: "Silicone" } }, { key: { text: "Inserted" }, value: { text: "2018" } } ] }) }} {% endcall %} 
{% raw %}{% call appSummaryCard({
  title: { text: "Implant — left breast" },
  actions: {
    items: [
      { text: "Edit", href: "#", visuallyHiddenText: "implant details" }
    ]
  }
}) %}
  ...
{% endcall %}{% endraw %}

With multiple actions

{% call appSummaryCard({ title: { text: "Biopsy — right breast" }, actions: { items: [ { text: "Edit", href: "#", visuallyHiddenText: "biopsy record" }, { text: "Delete", href: "#", visuallyHiddenText: "biopsy record" } ] } }) %} {{ appSummaryList({ rows: [ { key: { text: "Date" }, value: { text: "March 2021" } }, { key: { text: "Outcome" }, value: { text: "Benign" } } ] }) }} {% endcall %}

Custom heading level

Use title.headingLevel to change the heading element (default: h2).

{% call appSummaryCard({ title: { text: "Previous mammogram", headingLevel: 3 } }) %}

This card uses an h3 title.

{% endcall %} 
{% raw %}{% call appSummaryCard({
  title: { text: "Previous mammogram", headingLevel: 3 }
}) %}
  ...
{% endcall %}{% endraw %}
