Autocomplete

Accessible autocomplete built on accessible-autocomplete. Progressive enhancement from <select> or <input>. Uses an improved sort algorithm from DFE that scores exact matches higher, ignores stop words, and supports synonyms.

Macro: appAutocomplete(params)

Params reference

{% raw %}{{ appAutocomplete({

  {# --- Core --- #}
  id: "...",           {# string; defaults to name #}
  name: "...",         {# string; REQUIRED #}
  type: "select",      {# "select" (default) or "input" #}
  value: "...",        {# pre-selected value #}

  {# --- Labels and messages --- #}
  label: {             {# REQUIRED #}
    text: "...",
    isPageHeading: false,
    classes: "...",
    size: "s|m|l|xl"
  },
  hint: { text: "..." },
  errorMessage: { text: "..." },
  formGroup: { classes: "..." },
  classes: "...",      {# extra classes on the select/input element #}
  attributes: {},      {# extra HTML attributes on the select/input element #}

  {# --- Behaviour (all optional, override library defaults only when set) --- #}
  showAllValues: false,      {# show all options on focus + dropdown arrow #}
  autoselect: false,         {# highlight first result as you type; disable with showAllValues #}
  minLength: 0,              {# min chars before suggestions appear #}
  confirmOnBlur: true,       {# confirm selection on blur #}
  showNoOptionsFound: true,  {# show "No results found" message #}

  {# --- Select variant (type: "select") --- #}
  items: [
    {
      value: "...",        {# submitted value #}
      text: "...",         {# display text #}
      synonyms: ["..."],   {# alternative search terms #}
      hint: "...",         {# secondary line in dropdown #}
      append: "...",       {# text after name in dropdown #}
      boost: 1,            {# relevance multiplier #}
      disabled: false
    }
  ],

  {# --- Input variant (type: "input") --- #}
  source: [
    "Plain string",                       {# or ... #}
    { text: "...", hint: "...", append: "...", synonyms: [], boost: 1 }
  ]

}) }}{% endraw %}

Select (default)

Submitted value is the option's value attribute. Without JS a standard <select> is shown.

{{ appAutocomplete({ id: "country", name: "country", value: data.country, label: { text: "Country or territory" }, items: [ { value: "eng", text: "England", synonyms: ["UK", "United Kingdom", "Britain", "Great Britain"] }, { value: "sco", text: "Scotland", synonyms: ["UK", "United Kingdom", "Britain"] }, { value: "wal", text: "Wales", synonyms: ["UK", "United Kingdom", "Britain"] }, { value: "nir", text: "Northern Ireland", synonyms: ["UK", "United Kingdom"] }, { value: "fra", text: "France" }, { value: "deu", text: "Germany" }, { value: "irl", text: "Ireland" }, { value: "nld", text: "Netherlands" }, { value: "esp", text: "Spain" }, { value: "swe", text: "Sweden" } ] }) }}

With hint, append, and item hints

hint adds field-level hint text. append on an item shows extra text after the name in the dropdown. hint on an item shows a secondary line.

{{ appAutocomplete({ id: "screening-unit", name: "screeningUnit", label: { text: "Screening unit" }, hint: { text: "Start typing a unit name or location." }, items: [ { value: "bsu-01", text: "City Breast Screening Unit", append: "BSU-01", hint: "City Hospital, EC1A 1BB" }, { value: "bsu-02", text: "Northfield Breast Screening Unit", append: "BSU-02", hint: "Northfield General, N1 9GU" }, { value: "bsu-03", text: "Riverside Breast Screening Unit", append: "BSU-03", hint: "Riverside NHS Trust, SE1 7PB" } ] }) }} 
{% raw %}{{ appAutocomplete({
  id: "screening-unit",
  name: "screeningUnit",
  label: { text: "Screening unit" },
  hint: { text: "Start typing a unit name or location." },
  items: [
    { value: "bsu-01", text: "City Breast Screening Unit", append: "BSU-01", hint: "City Hospital, EC1A 1BB" },
    ...
  ]
}) }}{% endraw %}

Show all values

showAllValues: true shows all options on focus and adds a dropdown arrow. Use with autoselect: false — otherwise the first item is pre-highlighted when the list opens.

{{ appAutocomplete({ id: "risk-level", name: "riskLevel", label: { text: "Risk level" }, showAllValues: true, autoselect: false, items: [ { value: "routine", text: "Routine" }, { value: "moderate", text: "Moderate risk" }, { value: "high", text: "High risk" }, { value: "very-high", text: "Very high risk" } ] }) }} 
{% raw %}{{ appAutocomplete({
  id: "risk-level",
  name: "riskLevel",
  label: { text: "Risk level" },
  showAllValues: true,
  autoselect: false,
  items: [...]
}) }}{% endraw %}

Input variant

type: "input" renders a free-text input. Submitted value is whatever the user types — not constrained to the list. Source items can be plain strings or objects with text, hint, append, synonyms, boost.

{% set hospitalSource = [ { text: "City Hospital", hint: "EC1A 1BB" }, { text: "County General Hospital", hint: "N1 9GU" }, { text: "Memorial Hospital", hint: "SE1 7PB", synonyms: ["Mem"] }, "Royal Infirmary", "St Mary's Hospital", "University Hospital", "Western General" ] %}
{{ appAutocomplete({ id: "hospital", name: "hospital", type: "input", value: data.hospital, label: { text: "Hospital name" }, source: hospitalSource }) }}
{% if data.hospital %}

Submitted value: {{ data.hospital }}

{% endif %}

}) }}{% endraw %}
