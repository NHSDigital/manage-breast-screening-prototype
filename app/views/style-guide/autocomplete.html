Autocomplete

An accessible autocomplete built on accessible-autocomplete. Add data-module="app-autocomplete" to a <select> or <input> to enhance it.

Uses an improved sort algorithm (adapted from DFE Register Trainee Teachers) that prioritises closer matches, matches all words in any order, ignores punctuation and stop words (the, of, in…), and supports synonyms and boosts via data attributes.

Both examples below use a form so you can verify the submitted value is correct.

Enhancing a select element

The select is progressively enhanced. Without JavaScript, a standard dropdown is shown. With JavaScript, the select is replaced with an autocomplete input that filters the options.

The submitted value is the option value (not the display text), matching the original select behaviour.

Enhancing an input element

An <input> can also be enhanced by providing a data-source attribute containing a JSON array of strings. The submitted value is whatever the user types or selects.
