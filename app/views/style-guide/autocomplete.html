Autocomplete

An accessible autocomplete built on accessible-autocomplete. Progressively enhanced from a <select> or <input>.

Uses an improved sort algorithm (adapted from DFE Register Trainee Teachers) that prioritises closer matches, matches all words in any order, ignores punctuation and stop words, and supports synonyms via data-synonyms attributes.

Macro: appAutocomplete(params)

Select-based (default)

Renders a <select> progressively enhanced into an autocomplete. The submitted value is the option's value attribute, not its display text. Without JavaScript a standard dropdown is shown.

{{ appAutocomplete({ id: "country", name: "country", value: data.country, label: { text: "Country or territory" }, items: [ { value: "eng", text: "England", synonyms: ["UK", "United Kingdom", "Britain", "Great Britain"] }, { value: "sco", text: "Scotland", synonyms: ["UK", "United Kingdom", "Britain"] }, { value: "wal", text: "Wales", synonyms: ["UK", "United Kingdom", "Britain"] }, { value: "nir", text: "Northern Ireland", synonyms: ["UK", "United Kingdom"] }, { value: "fra", text: "France" }, { value: "deu", text: "Germany" }, { value: "irl", text: "Ireland" }, { value: "nld", text: "Netherlands" }, { value: "esp", text: "Spain" }, { value: "swe", text: "Sweden" } ] }) }}
{% if data.country %}

Submitted value: {{ data.country }}

{% endif %} 
{% raw %}{{ appAutocomplete({
  id: "country",
  name: "country",
  value: data.country,
  label: { text: "Country or territory" },
  items: [
    { value: "eng", text: "England", synonyms: ["UK", "United Kingdom", "Britain"] },
    { value: "sco", text: "Scotland" },
    ...
  ]
}) }}{% endraw %}

With hint, append, and item hints

Use hint for field-level hint text. Use append on items to show extra text after the option name in the dropdown. Use hint on items to show a secondary line of text.

{{ appAutocomplete({ id: "screening-unit", name: "screeningUnit", label: { text: "Screening unit" }, hint: { text: "Start typing a unit name or location." }, items: [ { value: "bsu-01", text: "City Breast Screening Unit", append: "BSU-01", hint: "City Hospital, EC1A 1BB" }, { value: "bsu-02", text: "Northfield Breast Screening Unit", append: "BSU-02", hint: "Northfield General, N1 9GU" }, { value: "bsu-03", text: "Riverside Breast Screening Unit", append: "BSU-03", hint: "Riverside NHS Trust, SE1 7PB" } ] }) }} 
{% raw %}{{ appAutocomplete({
  id: "screening-unit",
  name: "screeningUnit",
  label: { text: "Screening unit" },
  hint: { text: "Start typing a unit name or location." },
  items: [
    { value: "bsu-01", text: "City Breast Screening Unit", append: "BSU-01", hint: "City Hospital, EC1A 1BB" },
    ...
  ]
}) }}{% endraw %}

Show all values

Set showAllValues: true to show all options when the field is focused, like a dropdown. Adds a dropdown arrow. Typing still filters the list.

Set autoselect: false alongside it — otherwise the first option is pre-highlighted when the list opens, which is usually unwanted in dropdown mode.

{{ appAutocomplete({ id: "risk-level", name: "riskLevel", label: { text: "Risk level" }, showAllValues: true, autoselect: false, items: [ { value: "routine", text: "Routine" }, { value: "moderate", text: "Moderate risk" }, { value: "high", text: "High risk" }, { value: "very-high", text: "Very high risk" } ] }) }} 
{% raw %}{{ appAutocomplete({
  id: "risk-level",
  name: "riskLevel",
  label: { text: "Risk level" },
  showAllValues: true,
  autoselect: false,
  items: [...]
}) }}{% endraw %}

Input-based variant

Set type: "input" to render a free-text input with suggestions from a source array. The submitted value is whatever the user types or selects — not constrained to the list.

Source items can be plain strings, or objects with text, hint, append, synonyms, and boost — the same shape as select items.

{% set hospitalSource = [ { text: "City Hospital", hint: "EC1A 1BB" }, { text: "County General Hospital", hint: "N1 9GU" }, { text: "Memorial Hospital", hint: "SE1 7PB", synonyms: ["Mem"] }, "Royal Infirmary", "St Mary's Hospital", "University Hospital", "Western General" ] %}
{{ appAutocomplete({ id: "hospital", name: "hospital", type: "input", value: data.hospital, label: { text: "Hospital name" }, source: hospitalSource }) }}
{% if data.hospital %}

Submitted value: {{ data.hospital }}

{% endif %} 
{% raw %}{{ appAutocomplete({
  id: "hospital",
  name: "hospital",
  type: "input",
  value: data.hospital,
  label: { text: "Hospital name" },
  source: [
    { text: "City Hospital", hint: "EC1A 1BB" },
    "Royal Infirmary",
    ...
  ]
}) }}{% endraw %}
