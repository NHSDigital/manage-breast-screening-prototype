Notification banner

Wraps the NHS notification banner with a compact (inline) variant. Use appNotificationBanner in place of the standard notificationBanner when you need compact mode support.

Macro: appNotificationBanner(params)

Set size: "compact" (or compact: true) to use the compact template. Otherwise renders the standard NHS banner.

Important / info (default)

Compact banners

Compact banners are slimmer and suited for inline contextual alerts — for example, showing the status of a record within a larger page.

Important / info

{% raw %}{{ appNotificationBanner({

  text: "..."
