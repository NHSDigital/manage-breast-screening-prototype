Option picker

An inline group of radio (or checkbox) options styled as buttons or tags, replacing a select element with a single-click interaction. No JavaScript required.

Under the hood this uses standard <input type="radio"> markup with visually-hidden inputs. The label is styled to look like a button or tag and its selected state is driven by the CSS :checked selector.

Button variant

Each option looks like an NHS secondary button. The selected option fills with the brand blue.
