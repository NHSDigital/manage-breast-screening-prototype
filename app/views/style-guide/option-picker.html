Option picker

An inline group of radio (or checkbox) options styled as buttons, replacing a select element with a single-click interaction. No JavaScript required.

Under the hood this uses standard <input type="radio"> markup with visually-hidden inputs. The label is styled to look like a button and its selected state is driven by the CSS :checked selector.

Radio (default)

Icons default to on — a radio dot appears inside each option.

{{ appOptionPicker({ name: "sort-demo-1", value: "desc", fieldset: { legend: { text: "Sort order" } }, items: [ { value: "asc", text: "Ascending" }, { value: "desc", text: "Descending" } ] }) }}

Radio — more options, no icons

Set showIcons: false to hide the radio dot.

{{ appOptionPicker({ name: "sort-demo-2", value: "all", showIcons: false, fieldset: { legend: { text: "Image status" } }, items: [ { value: "all", text: "All" }, { value: "unread", text: "Unread" }, { value: "read", text: "Read" }, { value: "arbitrated", text: "Arbitrated" } ] }) }}

Radio — joined

Set joined: true to remove gaps between options. Only the outer corners are rounded.

{{ appOptionPicker({ name: "sort-demo-9", value: "desc", joined: true, showIcons: false, fieldset: { legend: { text: "Sort order" } }, items: [ { value: "asc", text: "Ascending" }, { value: "desc", text: "Descending" } ] }) }}
