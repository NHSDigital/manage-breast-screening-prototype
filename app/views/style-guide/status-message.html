Status message

A bold inline status line with an optional icon. Used to show the current state of something — for example, an opinion in the reading workflow.

Macro: appStatusMessage(params)

Colour variants: blue, green, red. Size variant: small (via classes).

No colour (default)

With icons

Pass an icon name to icon. Icons are rendered via appIcon.

{{ appStatusMessage({ text: "Recall recommended", colour: "red", icon: "alert" }) }} {{ appStatusMessage({ text: "Routine recall", colour: "blue", icon: "information" }) }} {{ appStatusMessage({ text: "Normal — no recall", colour: "green", icon: "tick" }) }} {{ appStatusMessage({ text: "Technical repeat needed", colour: "red", icon: "warning" }) }}
