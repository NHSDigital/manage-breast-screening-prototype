Button menu

A toggle button that reveals a group of secondary action buttons. Uses data-module="app-button-menu" to wire up the toggle behaviour.

Macro: buttonMenu(params)

Default

The primary visible button is configured via params.button. Menu items are listed in params.items.

{{ buttonMenu({ button: { text: "Actions" }, items: [ { text: "Edit", href: "#" }, { text: "Duplicate", href: "#" }, { text: "Delete", href: "#" } ] }) }} 
{% raw %}{{ buttonMenu({
  button: { text: "Actions" },
  items: [
    { text: "Edit", href: "#" },
    { text: "Duplicate", href: "#" },
    { text: "Delete", href: "#" }
  ]
}) }}{% endraw %}

Fewer items

{{ buttonMenu({ button: { text: "More options" }, items: [ { text: "Reschedule", href: "#" }, { text: "Cancel appointment", href: "#" } ] }) }} 
{% raw %}{{ buttonMenu({
  button: { text: "More options" },
  items: [
    { text: "Reschedule", href: "#" },
    { text: "Cancel appointment", href: "#" }
  ]
}) }}{% endraw %}

Right-aligned menu

Use alignMenu: "right" to align the dropdown to the right edge of the button.

{{ buttonMenu({ button: { text: "Actions" }, alignMenu: "right", items: [ { text: "Edit", href: "#" }, { text: "Archive", href: "#" } ] }) }} 
{% raw %}{{ buttonMenu({
  button: { text: "Actions" },
  alignMenu: "right",
  items: [...]
}) }}{% endraw %}
