Secondary navigation

Tab-style navigation for moving between sections of a page or workflow. Typically placed in the pageNavigation block so it spans the full width above the main content area.

Macro: appSecondaryNavigation(params)

Items can be current (active page, adds aria-current="page"), active (visually bold but not the current page), or disabled.

Basic with current item

{{ appSecondaryNavigation({ items: [ { text: "Appointment", href: "#", current: true }, { text: "Participant", href: "#" }, { text: "Medical information", href: "#" } ] }) }} 
{% raw %}{{ appSecondaryNavigation({
  items: [
    { text: "Appointment", href: "#", current: true },
    { text: "Participant", href: "#" },
    { text: "Medical information", href: "#" }
  ]
}) }}{% endraw %}

With disabled item

{{ appSecondaryNavigation({ items: [ { text: "Appointment", href: "#" }, { text: "Participant", href: "#", current: true }, { text: "Images", href: "#", disabled: true } ] }) }} 
{% raw %}{ text: "Images", href: "#", disabled: true }{% endraw %}

Active but not current

Use active: true to show a tab as bold (for example, when its sub-section is active) without marking it as the current page.

{{ appSecondaryNavigation({ items: [ { text: "Appointment", href: "#", active: true }, { text: "Participant", href: "#" }, { text: "Medical information", href: "#" } ] }) }} 
{% raw %}{ text: "Appointment", href: "#", active: true }{% endraw %}

Custom accessible label

Use visuallyHiddenTitle to provide a descriptive aria-label for the <nav> element.

{{ appSecondaryNavigation({ visuallyHiddenTitle: "Appointment sections", items: [ { text: "Summary", href: "#", current: true }, { text: "Details", href: "#" } ] }) }} 
{% raw %}{{ appSecondaryNavigation({
  visuallyHiddenTitle: "Appointment sections",
  items: [...]
}) }}{% endraw %}
