Custom components and utilities built for this prototype that are not part of the NHS design system.

Components

  • Autocomplete — an accessible autocomplete input that enhances a select or free-text input with filtered suggestions
  • Button menu — a toggle button that reveals a group of secondary actions
  • Count — a small badge used to show a number alongside a label
  • Details — an extended version of the NHS details component with subtitle, contents summary, and status tag
  • Forward link — a directional link or button pointing forward
  • Icons — custom icons not found in the NHS icon set
  • Modal — a modal dialog for confirmations and actions
  • Notification banner — the NHS notification banner extended with a compact/inline variant
  • Option picker — an inline group of radio options styled as buttons or tags, replacing a select with a single click
  • Secondary navigation — tab-style navigation within a section
  • Status bar — a sticky full-width bar showing contextual key/value information
  • Status message — a bold status line with optional icon and colour variants
  • Stepper input — a number input with increment and decrement buttons
  • Summary card — a summary card with a title and freeform content area
  • Summary list — an extended summary list that accepts HTML values and supports caller content

Utilities

  • Typography & CSS utilities — button-link, error/warning links, secondary text, inset text compact
