Summary list

An extended version of the NHS summary list. When all values are plain text, the macro falls through to the standard NHS component. The custom template is used when any row uses row.html for a value, or when using the caller() pattern.

Macros: appSummaryList(params), appSummaryListRow(row)

Standard (falls through to NHS component)

When all values are plain text, this is identical to the NHS summaryList.

{{ appSummaryList({ rows: [ { key: { text: "Name" }, value: { text: "Janet Williams" } }, { key: { text: "Date of birth" }, value: { text: "12 March 1965" } }, { key: { text: "NHS number" }, value: { text: "485 777 3456" } } ] }) }} 
{% raw %}{{ appSummaryList({
  rows: [
    { key: { text: "Name" }, value: { text: "Janet Williams" } },
    { key: { text: "Date of birth" }, value: { text: "12 March 1965" } }
  ]
}) }}{% endraw %}

With HTML values

Use row.html instead of row.value when the value needs to contain HTML — for example, a tag or a list.

{% set statusTagHtml %} Checked in {% endset %}
{{ appSummaryList({ rows: [ { key: { text: "Status" }, html: statusTagHtml }, { key: { text: "Appointment" }, value: { text: "10:30, 31 March 2026" } }, { key: { text: "Screening room" }, html: "Room 2" } ] }) }} 
{% raw %}{ key: { text: "Status" }, html: "Checked in" }{% endraw %}

Using appSummaryListRow

Use appSummaryListRow to build individual rows independently — useful when rows are generated conditionally.

{{ appSummaryListRow({ key: { text: "Left breast" }, value: { text: "Normal" } }) }} {{ appSummaryListRow({ key: { text: "Right breast" }, html: "Recall" }) }} 
{% raw %}

  {{ appSummaryListRow({ key: { text: "Left breast" }, value: { text: "Normal" } }) }}
  {{ appSummaryListRow({ key: { text: "Right breast" }, html: "..." }) }}

{% endraw %}
