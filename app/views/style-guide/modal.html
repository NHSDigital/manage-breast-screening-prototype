Modal

A modal dialog for confirmations and other inline decisions. Toggled by calling openModal('modal-id') from JavaScript. Closed by any action button with data-modal-action="close".

Macros: appModal(params), appConfirmationModal(params)

Confirmation modal (preset)

appConfirmationModal provides a standard two-button confirm/cancel layout.

{{ button({ text: "Open confirmation modal", classes: "nhsuk-button--secondary", attributes: { onclick: "openModal('demo-confirm'); return false;" } }) }} {{ appConfirmationModal({ id: "demo-confirm", title: "Are you sure?", content: "

This action cannot be undone. The record will be permanently removed.

", confirmText: "Yes, delete", cancelText: "Cancel" }) }}

  id: "demo-confirm",
  title: "Are you sure?",

...
",
  confirmText: "Yes, delete",
  cancelText: "Cancel"
}) }}{% endraw %}

Custom actions

Use appModal with a custom actions array for more control. Actions can be buttons (element: "button") or links (element: "link").

{{ button({ text: "Open custom modal", classes: "nhsuk-button--secondary", attributes: { onclick: "openModal('demo-custom'); return false;" } }) }} {{ appModal({ id: "demo-custom", title: "Choose an action", content: "

Select what you would like to do with this appointment.

", actions: [ { text: "Reschedule", element: "button", action: "close" }, { text: "Cancel appointment", element: "button", classes: "nhsuk-button--secondary", action: "close" }, { text: "Go back", element: "link", action: "close", href: "#" } ] }) }} 
{% raw %}{{ appModal({
  id: "demo-custom",
  title: "Choose an action",
  content: "
...
",
  actions: [
    { text: "Reschedule", element: "button", action: "close" },
    { text: "Cancel", element: "button", classes: "nhsuk-button--secondary", action: "close" },
    { text: "Go back", element: "link", action: "close", href: "#" }
  ]
}) }}{% endraw %}

Content-only modal

A modal with no title — content fills the dialog directly.

{{ button({ text: "Open content modal", classes: "nhsuk-button--secondary", attributes: { onclick: "openModal('demo-content'); return false;" } }) }} {{ appModal({ id: "demo-content", content: "

This modal has no title. The content area fills the full width of the dialog.

", actions: [ { text: "Close", element: "button", classes: "nhsuk-button--secondary", action: "close" } ] }) }} 
{% raw %}{{ appModal({
  id: "demo-content",
  content: "
...
",
  actions: [
    { text: "Close", element: "button", classes: "nhsuk-button--secondary", action: "close" }
  ]
}) }}{% endraw %}
