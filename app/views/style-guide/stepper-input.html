Stepper input

A number input with increment (＋) and decrement (−) buttons. Requires JavaScript; the plain number input is shown without JavaScript (progressive enhancement).

Macro: appStepperInput(params)

Basic

{{ appStepperInput({ id: "images-count", name: "imagesCount", value: 4, label: { text: "Number of images" } }) }} 
{% raw %}{{ appStepperInput({
  id: "images-count",
  name: "imagesCount",
  value: 4,
  label: { text: "Number of images" }
}) }}{% endraw %}

With hint

{{ appStepperInput({ id: "repeat-count", name: "repeatCount", value: 1, label: { text: "Repeats needed" }, hint: { text: "Enter the total number of images to retake." } }) }} 
{% raw %}{{ appStepperInput({
  id: "repeat-count",
  name: "repeatCount",
  value: 1,
  label: { text: "Repeats needed" },
  hint: { text: "Enter the total number of images to retake." }
}) }}{% endraw %}

With min and max

Use min and max to constrain the value. The decrement button is disabled when the current value equals the minimum, and the increment button is disabled when it equals the maximum.

{{ appStepperInput({ id: "room-capacity", name: "roomCapacity", value: 1, min: 1, max: 10, label: { text: "Rooms available" } }) }} 
{% raw %}{{ appStepperInput({
  id: "room-capacity",
  name: "roomCapacity",
  value: 1,
  min: 1,
  max: 10,
  label: { text: "Rooms available" }
}) }}{% endraw %}

With non-default step

Use step to change the increment amount (default is 1).

{{ appStepperInput({ id: "exposure-time", name: "exposureTime", value: 10, step: 5, min: 0, label: { text: "Exposure (ms)" } }) }} 
{% raw %}{{ appStepperInput({
  id: "exposure-time",
  name: "exposureTime",
  value: 10,
  step: 5,
  min: 0,
  label: { text: "Exposure (ms)" }
}) }}{% endraw %}
